    How Fireworks is useful

    • CAN slice parts when linking
    • Gridlines help match lines
    • Slicing helps us know which part is needed or useful and which are not
    • Can make buttons that could be put into dreamweaver

    How Fireworks is not useful

    • All the slices are in images
    • Unlike DW we need to slice the parts to link each part
    • Canvas size is limited
  • Dreamweaver Version

    How Dreamweaver is useful

    • Automatically finishes HTML code for us
    • It is focused more on the coding
    • CSS rules on the side
    • Can split view - It provides both the design so we know how it looks and also provides the code so it also gives us an idea of how the code looks like.

    How Dreamweaver is not useful

    • Unlike Fireworks where we can just choose our own fonts and colors and other elements, in dreamweaver we need to use CSS rules.
